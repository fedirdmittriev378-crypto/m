# Личный бюджет — полная версия

- Flask веб-приложение
- Валюта: RUB
- Категории операций
- Повторяющиеся платежи (daily/weekly/monthly)
- Отчёты по месяцам
- Цели (Goals) с прогрессом
- Импорт/экспорт CSV
- Простые графики (matplotlib)

Установка:
1. python -m venv venv
2. source venv/bin/activate  (или venv\Scripts\activate на Windows)
3. pip install -r requirements.txt
4. python run.py
5. Открыть http://127.0.0.1:5000
